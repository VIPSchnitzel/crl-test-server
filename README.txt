Hello dear user!
 
This server was only tested under Suse.
To run this server you might need to execute

sudo zypper in openssl
sudo zypper in libevent-2_0-5

from project root: sudo dep/install-dep.sh

To build this application you need to execute
sudo zypper in gcc7
sudo zypper in libopenssl-devel
sudo zypper in libevent-devel

(or the eqiuvalent for your system)
